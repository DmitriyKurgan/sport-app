# FitTrack — Контекст проекта

## Описание

Персонализированное фитнес-приложение для генерации тренировочных программ, отслеживания прогресса и аналитики.

## Текущий статус

**Фаза: Проектирование архитектуры**

## Стек технологий

| Слой | Технология |
|------|-----------|
| Backend | NestJS + TypeORM |
| Database | PostgreSQL |
| Frontend | Next.js 14 (App Router) |
| State | Redux Toolkit + RTK Query |
| Charts | Recharts |
| Auth | JWT + Refresh tokens |
| Cache | Redis (опционально) |
| Queue | Bull (фоновые задачи) |

## Ключевые решения

1. **TypeORM вместо Prisma** — лучше интегрируется с NestJS, поддержка декораторов
2. **RTK Query вместо React Query** — единый стор Redux, встроенная нормализация кэша
3. **App Router (Next.js)** — server components, streaming, layouts
4. **JWT + Refresh** — стандартный подход, refresh через httpOnly cookie
